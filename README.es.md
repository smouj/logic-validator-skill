# Validador de Lógica

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad de Validador de Lógica para OpenClaw.

## Qué Soluciona
Valida la lógica del código e identifica inconsistencias lógicas.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque de seguridad primero
- Soporte de reversión

## Uso
/logic-validator

## Ejemplo
```
Analyze code for quality assurance issues.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
